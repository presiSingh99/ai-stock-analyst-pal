# AI Stock Analyst Copilot

## Overview
AI Financial Analyst Copilot is a publicly deployed AI-powered financial dashboard that combines live market data with Large Language Models to generate analyst-style insights.

## Features
- Live stock data via Yahoo Finance
- Multi-ticker comparison
- KPI dashboards
- Moving averages and trend analysis
- AI-generated analyst summaries using OpenAI
- Natural language questions
- Public deployment using Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Yahoo Finance
- Pandas
- NumPy
- OpenAI API
- GitHub
- Streamlit Cloud

## Architecture
User → Streamlit → Yahoo Finance → KPI Engine → OpenAI → AI Summary
